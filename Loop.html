Basically, most of the logic regarding for-loop and while-loop are same except for some minor difference in syntax in while-loop.
However, the for-loop in Python can be flexible. As it can directly use each item in different arrays i.e. List, Tuple, Dictionary.
For for-loop in Python, there no initialization action after one loop inside the condition which these action have to be write in the codes that JavaScript's for-loop will be clear and easy to understand.
